# <a name="ltcinttypesgt"></a>&lt;cinttypes&gt;

標準 C ライブラリヘッダーが含まれてい \<inttypes.h> ます。 このヘッダーを含めるには、も含まれ \<cstdint> ます。

## <a name="syntax"></a>構文

```cpp
#include <cinttypes>
```

## <a name="remarks"></a>解説

また、このヘッダーをインクルードすると、標準 C ライブラリ ヘッダーの外部リンケージで宣言された名前が、`std` 名前空間でも宣言されます。
