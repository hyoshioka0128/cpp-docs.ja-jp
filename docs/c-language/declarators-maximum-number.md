# <a name="declarators-maximum-number"></a>宣言子:最大数

**ANSI 3.5.4** 演算型、構造体型、または共用体型を変更できる宣言子の最大数

Microsoft C では、宣言子の数は制限されません。 数は、使用できるメモリ容量によってのみ制限されます。
