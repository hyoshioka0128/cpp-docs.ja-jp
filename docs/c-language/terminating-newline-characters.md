# <a name="terminating-newline-characters"></a>改行文字の終了

**ANSI 4.9.2** テキスト ストリームの最後の行が、終了の改行文字を必要とするかどうか

ストリーム関数は、新しい行またはファイルの末尾を行の終端文字として認識します。
