# <a name="compiler-error-c2406"></a>コンパイラ エラー C2406

'identifier': 'context' で名前が定義されていません

未定義の識別子が `SIZE` 、、 `LENGTH` 、またはメンバー選択 (.) 演算子と共に使用されています。
