# <a name="fatal-error-c1351"></a>致命的なエラー C1351

> dll '*dll*' の読み込みエラーです: 互換性のないバージョンです。

間違ったバージョンの DLL が見つかりました。

これは、インストールに問題があることを示しているため、Visual Studio を再インストールする必要があります。
