# <a name="compiler-warning-level-1-c4344"></a>コンパイラの警告 (レベル 1) C4344

動作の変更: 明示的なテンプレート引数を使用すると、' function ' が呼び出されることになります。

明示的なテンプレート引数を使用した関数の呼び出しは、明示的な引数が指定されていない場合とは異なる関数を呼び出します。
