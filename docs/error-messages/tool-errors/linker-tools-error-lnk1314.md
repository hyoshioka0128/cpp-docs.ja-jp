# <a name="linker-tools-error-lnk1314"></a>リンカ ツール エラー LNK1314

COFF シンボルテーブルが壊れているか、無効です (定義されていない静的シンボルまたはラベルシンボル)

指定されたオブジェクトのセクションに対してコンパイラによって生成されたコンテンツが破損している可能性があります。

このエラーは、コンパイラのエラーを表している可能性があります。
