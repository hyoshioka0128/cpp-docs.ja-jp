# <a name="expression-evaluator-error-cxx0031"></a>式エバリュエーター エラー CXX0031

展開されていない式

式エバリュエーターで内部エラーが発生しました。

適切に評価できる同等の式を記述することもできます。

このエラーは CAN0031 と同じです。
