# <a name="linker-tools-error-lnk1332"></a>リンカ ツール エラー LNK1332

リンカーは、現在のターゲットを生成したときに、 `count` それぞれが1つのモジュールにインポートされ、別のモジュールでも定義されている Windows ランタイムの型> <を検出しました。
