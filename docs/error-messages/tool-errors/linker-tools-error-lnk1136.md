# <a name="linker-tools-error-lnk1136"></a>リンカ ツール エラー LNK1136

ファイルが無効または壊れています

入力ファイルのヘッダーが壊れているか、サイズがゼロか、異常に小さいかのいずれかです。

また、ファイルが追加の依存関係として誤って指定されている場合にも、LNK1136 が発生する可能性があります。
