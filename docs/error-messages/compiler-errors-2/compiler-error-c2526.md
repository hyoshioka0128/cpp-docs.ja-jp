# <a name="compiler-error-c2526"></a>コンパイラ エラー C2526

' identifier1 ': C リンケージ関数は、C++ クラス ' identifier2 ' を返すことはできません

C リンケージで定義された関数は、ユーザー定義型を返すことができません。
