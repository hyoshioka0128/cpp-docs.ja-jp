# <a name="compiler-error-c3159"></a>コンパイラエラー C3159

' pointer ': 値型へのポインターの配列を宣言することはできません

値型へのポインターの配列を宣言することはできません。

C3159 は、互換性のために残されているコンパイラオプション **/clr: oldSyntax** を使用してのみ到達可能です。
